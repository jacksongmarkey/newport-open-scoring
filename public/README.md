# Newport Open — Live Scoring

Real-time scoring app for The Newport Open golf tournament. Teams enter their hole-by-hole scores; the leaderboard updates live for spectators via Server-Sent Events.

## Tech Stack

- **Backend:** Node.js + Express
- **Live updates:** Server-Sent Events (SSE)
- **Storage:** JSON file on persistent disk
- **Hosting:** Render.com (free tier)

## Local Development

```bash
npm install
npm start
# open http://localhost:3000
```

## Deploy to Render.com

This repo includes `render.yaml` for one-click deployment via Render Blueprints.

1. Push this folder to a GitHub repository
2. Go to [render.com](https://render.com) → New → Blueprint
3. Connect your GitHub repo → Render reads `render.yaml` automatically
4. Click **Apply** — Render creates the service + persistent disk
5. Your app will be live at `https://newport-open-scoring.onrender.com`

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/` | Leaderboard UI (HTML page) |
| GET | `/api/state` | Current state (teams + scores + pars) |
| POST | `/api/teams` | Register a team `{name, players}` |
| POST | `/api/score` | Submit score `{teamId, hole, score}` (hole is 0-indexed) |
| DELETE | `/api/reset` | Wipe all scores (requires `X-Admin-Token` header) |
| GET | `/events` | SSE stream for live leaderboard updates |
| GET | `/healthz` | Health check for Render |

## Admin Reset

To wipe all scores after the tournament:

```bash
curl -X DELETE https://newport-open-scoring.onrender.com/api/reset \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"
```

The `ADMIN_TOKEN` env var is auto-generated by Render. View it in:
**Render Dashboard → newport-open-scoring → Environment → ADMIN_TOKEN**

## Course Par

Edit the `PARS` array at the top of `server.js` to match Meadow Brook CC's scorecard if it differs from the default par 72.
